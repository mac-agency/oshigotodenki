お仕事電気について

お仕事電気とは

普段より多くの電気をお使いになる法人様や個人事業主様向けにお安く電力をご提供するプランです。
毎月の電力料金を抑えて経費を抑えて利益を向上させる、お仕事で使う電気の料金プランの決定版です。

Point

電気料金の削減されたい方にこそ
おススメの3つのポイント

  • 料金がお得

    現在の使用量が300kwh/月以上ご利用のお役様へ電気代が安くなるプランをご用意

  • 品質変わらず

    切替えても今までの設備をそのまま使用するので従来通りの品質でご利用いただけます

  • 安心をお届け

    停電時に実際に復旧作業にあたるのは地域の電力会社になりますので安心してご利用いただけます。

Liberalization

電力の自由化とは

低圧で電気の供給を受ける場合、契約できる電力会社は決まっていました。 2016年4月　全面自由化により複数の電力会社からお客様ご自身で選べるようになりました。
“生活に欠かせない電気”だからこそ お客様にあったプラン・サービスをご提案 ご提供いたします。

Q&A

よくある質問

☆切替についてのよくある質問

Q　電力会社を変更すると、新しい電線が事務所や店舗に引かれることになるのですか？

現在の一般送配電事業者が管理・運営する既存の送電線を経由し、電気が送られてきますので新しく事務所や店舗に電線が引かれることはございません。

Q　工事は必要ですか？

必要ありません。
ただし、スマートメーターが設置されていない場合は、一般電気事業者がスマートメーターを設置いたします。

Q　切り替えると停電しやすくなる等、電力供給の品質への影響があるのでしょうか？

今までと同様の送電線を経由して電力供給が行われるため、停電の起こる可能性や電気の品質は今までと変わりません。

Q　電気を切り替える時には、一時的に停電しますか？

スマートメーターへの取替工事が必要となる場合、一時的にブレーカーを落とすなど、停電する場合があります。切り替えのお手続きの際に、電力会社へご確認ください。

Q　停電が起きた場合はどこに問い合わせをすれば良いですか？

停電の原因には、ご利用場所敷地内の電気設備（内線）側に原因がある場合と、送配電設備（外線）側に原因がある場合がございます。 一般送配電事業者へ直接ご確認ください。一時受付として弊社サポートセンターへお問い合わせいただくことも可能です。

Q　なぜ電気料金を安くできるんですか？

発電事業者から安く電力を調達し、設備投資等にかかるコストと、人件費等の固定費を抑制をしているので、従来の電気料金よりも安く提供することが可能となります。

Q　スマートメーターとはなんですか？また交換時立ち合いは必要ですか？

スマートメーターは、通信機能を有し、遠隔での検針等が可能となる新しい電力量計です。 　原則として、こちらで手続きを進めるため、お客様にお立合いいただくことはございません。 取り替え工事時に、一時的な停電が発生することがございますのでご了承ください。切り替えのお手続きの際に、電力会社へご確認ください。
※設置場所とご契約状況によってはお立合いいただく場合もございます

Q　お仕事でんきの契約期間は何年ですか？また、更新費用は発生しますか？

契約期間は2年でございます。更新費用は発生いたしません。

☆申込についてのよくある質問

Q　お仕事でんきの申込方法を教えてください。

お電話とWEBサイトから受け付けております。
お電話ですと、時間帯にもよりますがコールセンターの混み具合によりお待たせしてしまうこともございますのでWEBサイトからのご相談をおすすめ致します。
  WEBサイトからご相談の際電気の検針票をご用意いただき、必要事項を入力してください。ご希望の日時に折り返し、コールセンターよりお電話致します。

Q　申込後いつから切替りますか？

供給開始は検針日切り替えとなります。基本的にはお申し込みから半月経過した初回の検針日にて切り替えが可能となりますが、スイッチング処理などで次々回検針日の切り替えとなることがございます。 申し込み内容、および供給開始予定日はお申し込み完了、3営業日以内に会員証を郵送致しますので会員証記載の供給開始予定日をご確認願います。

Q　電力（動力200V）は申し込みはできますか？

もちろん可能です。電灯と動力の検針票をそれぞれ1枚ずつご準備の上ご相談ください。

Q　契約期間はありますか？また契約期間中に解約する場合、解約金はかかりますか？

契約期間は、供給開始日以降2年契約の自動更新となっております。
　2年以内にご解約の場合15000円の違約金がかかります。
※移転・引っ越しの場合は引き続きご継続いただければかかりません。

☆料金・お支払いについてよくある質問

Q　お支払方法にはどのようなものがありますか？

お支払いは、クレジットカードでの決済と、口座振替でのお支払となります。
ご利用可能なクレジットカード：Visa、MasterCard、American Express、DinersClub、JCB口座振替の場合、6日、23日、27日の振替となり、お客様の検針日により異なります。
口座振替の場合、通帳への記載は「EMI（デンキダイ）」、クレジットカードの明細への記載は「イイネデンリョク」or「デンキリョウキン」or「ENEでんき　東京都新宿区」のいずれかとなります。

Q　請求書や検針票は発行されますか？

お客様のマイページにて請求書を印刷することは可能です。電力・ガス会社からの検針票は発行されなくなりますが、マイページにてご使用量の確認ができます。

Q　お仕事でんきに切り替えると、毎月の電気代はどのくらい安くなりますか？

その月の使用量によって異なりますので詳しくは、重要事項説明書記載の価格表または料金シュミレーションにて詳細をご覧ください。
